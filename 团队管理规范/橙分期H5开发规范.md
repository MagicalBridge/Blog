## 1、每个页面对应的controller中所有依赖的服务,均使用单引号引入:
```js
// bad 模块名称 控制器名称 依赖服务 都应该使用单引号
angular.module("app.user")
  .controller("u.MyCtrl", [
    "$scope", "u.MyService",
    function($scope, mS) {}
  ]) 

// good
angular.module('app.user')
.controller('u.MyCtrl', [
  '$scope', 'u.MyService',
  function($scope, mS) {}
]) 
```

## 2、每个页面对应的controller中，对应的埋点逻辑对应的字段值，使用单引号
```js
// bad
sa.track('app_cfqa_pageview', {
  page_id: "040000000000",
  page_name: "用户-我的",
  page_comment: "用户我的页面"
});

// good
sa.track('app_cfqa_pageview', {
  page_id: '040000000000',
  page_name: '用户-我的',
  page_comment: '用户我的页面'
});
```
